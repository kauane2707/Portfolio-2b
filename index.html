O projeto utilizou um Arduino Uno para controlar um módulo de LEDs que simula um semáforo. O Arduino é programado para acender e apagar os LEDs em uma sequência específica, usando seus pinos digitais. Resistores são conectados aos LEDs para proteger o circuito, limitando a corrente elétrica que passa por eles.

Os principais materiais são uma placa Arduino Uno, um módulo de LEDs de semáforo (ou LEDs individuais) e resistores para proteção. Embora não visíveis, cabos jumper são usados para as conexões elétricas e um cabo USB conecta o Arduino ao computador para programação e energia. O software Arduino IDE é essencial para o desenvolvimento do código.

Este projeto é uma ferramenta didática para aprender programação e eletrônica básica com Arduino. Ele ensina o controle de LEDs e a criação de sequências temporizadas. Demonstra o princípio de funcionamento de semáforos reais, servindo como um protótipo simplificado.

o Arduino atua como um "controlador de tráfego" eletrônico.São escritos os códigos que ele deve seguir, e ele, por sua vez, envia os comandos elétricos para ligar e desligar os LEDs na sequência e nos tempos programados, simulando o funcionamento de um semáforo.

O projeto foi montado com um Arduino Uno, um sensor DHT (de temperatura e umidade) e uma protoboard. O sensor é conectado ao Arduino para alimentação (5V e GND) e para o envio de dados via um pino digital. As conexões são realizadas com fios jumper na protoboard, permitindo fácil montagem sem solda.

Os materiais essenciais incluem a placa Arduino Uno, o sensor DHT11 ou DHT22, e uma placa protoboard para as conexões. Fios jumper são usados para ligar os componentes, enquanto um cabo USB conecta o Arduino ao computador para programação e energia. É necessário o software Arduino IDE e uma biblioteca DHT para o código.

Este projeto serve para monitorar a temperatura e umidade de um ambiente. É um excelente recurso didático para aprender sobre eletrônica, programação com microcontroladores e interfaceamento de sensores. Ele forma uma base para desenvolver sistemas mais complexos, como automação residencial ou estações meteorológicas básicas.O Arduino, programado com um código específico, alimenta o sensor DHT. O sensor então mede a temperatura e umidade do ambiente e envia esses dados digitais para o Arduino. O microcontrolador processa essas informações, convertendo-as em valores legíveis que podem ser exibidos, por exemplo, no monitor serial do computador, em um ciclo contínuo de medições.
